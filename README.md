# 統計検定2級 CBT突破ドリル

統計検定2級で「公式問題集は解けるのに、CBT本番の初見問題で崩れる」状態を抜けるための、オリジナルWeb問題集です。

## 目的

- 暗記済みの公式問題と違う聞かれ方に慣れる
- 問題文から使う分布・推定・検定・回帰を選ぶ力を鍛える
- 不正解を分野別に残し、弱点モードで再演習する

## 使い方

`index.html` をブラウザで開くだけで使えます。GitHub Pagesに置く場合もビルド不要です。

## モード

- 演習: 分野と問題数を選んで解く
- 模試: 35問をランダムに解く
- 弱点: 過去に間違えた分野を優先して出題する

## 問題設計

現在は69種類の問題生成パターンを収録しています。単純計算だけでなく、CBTで差がつきやすい手法選択、標準誤差、p値、信頼区間、自由度、標本分布、過誤と検出力、回帰の解釈、カテゴリデータ、調査バイアス、数字からグラフの形を読む問題を含めています。

CBT実戦パターンでは、長めの事例文、表、A/Bテスト、クロス集計、対応のある前後比較、回帰出力、標本平均の分布、ヒストグラム・散布図・分布形状の読み取りなどを使い、問題文から必要な条件を抜き出す練習に寄せています。模試モードでは、CBT実戦・難問・判断問題が多めに出るようにしています。

各解説には「まず何を求める問題？」というTipsを付け、問題文を素人向けの言葉に翻訳してから、式と計算へ進む構成にしています。

## 公式範囲への対応

- 1変数データ: 中心傾向、散らばり、外れ値、変動係数、時系列、ローレンツ曲線・ジニ係数、ヒストグラム形状
- 2変数以上のデータ: 散布図と相関、カテゴリカルデータ、単回帰、予測、回帰係数の検定
- 推測のためのデータ収集法: 観察研究、標本調査法、調査バイアス、フィッシャーの3原則
- 確率: 条件付き確率、ベイズの定理、反復試行
- 確率分布: 二項、ポアソン、幾何、正規、t、カイ二乗、F分布の形状・用途
- 標本分布: 標本平均・標本比率、二項分布の正規近似、中心極限定理
- 推定: 一致性・不偏性、母平均・母比率・母分散、2標本平均差・比率差の区間推定
- 仮説検定: p値、2種類の過誤、母平均・母比率・母分散、対応あり/なし、F検定
- カイ二乗検定: 適合度検定、独立性の検定
- 線形モデル: 回帰分析、一元配置・二元配置分散分析、実験計画

## 参考にした学習範囲

添付PDFの章立てから、記述統計、確率統計、推測統計、仮説検定、線形モデル、適合度検定・独立性検定を主要分野として扱っています。

公式サイトの2026年5月時点の2級ページでは、CBT方式、4〜5肢選択、35問程度、90分、合格水準60点以上と案内されています。出題範囲は1変数データ、2変数以上のデータ、データ収集法、確率、確率分布、標本分布、推定、仮説検定、カイ二乗検定、線形モデルです。

公式問題や市販問題集の文章は収録していません。著作権を避けつつ、本番で必要な判断力を鍛えるため、数値や選択肢が変わるオリジナル問題として実装しています。

公式情報: https://www.toukei-kentei.jp/grade/grade2/

## 1ヶ月学習プラン

45点から60点以上を狙う前提では、知識の網羅よりも「落とす問題を減らす」方針で進めます。

- 1週目: 記述統計、グラフ、確率分布の形、標準化、条件付き確率を復習。演習モードで記述統計・確率・分布を重点的に解く。
- 2週目: 推定と検定の式選択を固める。信頼区間、標準誤差、p値、有意水準、片側・両側を毎日確認する。
- 3週目: 回帰、分散分析、カイ二乗検定、標本抽出、調査バイアスを演習。表から検定手法を選ぶ問題を増やす。
- 4週目: 模試モードを毎日1セット。間違えた問題は「計算ミス」「式選択ミス」「用語誤解」「問題文の読み落とし」に分類し、弱点モードで潰す。

毎日の配分は、基礎確認30分、Webドリル40分、間違いノート20分を目安にします。公式問題集は解き直しよりも、解いた後に「この問題は何を求める問題だったか」を1行で言い換えることを重視します。
