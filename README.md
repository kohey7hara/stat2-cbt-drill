# 統計検定2級 CBT突破ドリル

統計検定2級で「公式問題集は解けるのに、CBT本番の初見問題で崩れる」状態を抜けるための、オリジナルWeb問題集です。

## 目的

- 暗記済みの公式問題と違う聞かれ方に慣れる
- 問題文から使う分布・推定・検定・回帰を選ぶ力を鍛える
- 不正解を分野別に残し、弱点モードで再演習する

## 使い方

`index.html` をブラウザで開くだけで使えます。GitHub Pagesに置く場合もビルド不要です。

## モード

- 演習: 分野と問題数を選んで解く
- 模試: 35問をランダムに解く
- 弱点: 過去に間違えた分野を優先して出題する

## 問題設計

現在は43種類の問題生成パターンを収録しています。単純計算だけでなく、CBTで差がつきやすい手法選択、標準誤差、p値、信頼区間、自由度、標本分布、過誤と検出力、回帰の解釈、カテゴリデータ、調査バイアスを含めています。

各解説には「まず何を求める問題？」というTipsを付け、問題文を素人向けの言葉に翻訳してから、式と計算へ進む構成にしています。

## 参考にした学習範囲

添付PDFの章立てから、記述統計、確率統計、推測統計、仮説検定、線形モデル、適合度検定・独立性検定を主要分野として扱っています。

公式サイトの2026年5月時点の2級ページでは、CBT方式、4〜5肢選択、35問程度、90分、合格水準60点以上と案内されています。出題範囲は1変数データ、2変数以上のデータ、データ収集法、確率、確率分布、標本分布、推定、仮説検定、カイ二乗検定、線形モデルです。

公式問題や市販問題集の文章は収録していません。著作権を避けつつ、本番で必要な判断力を鍛えるため、数値や選択肢が変わるオリジナル問題として実装しています。

公式情報: https://www.toukei-kentei.jp/grade/grade2/

## GitHub Pages
